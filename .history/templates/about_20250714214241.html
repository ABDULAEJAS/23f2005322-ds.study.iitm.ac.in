About the Parking Management System

The Parking Management System is a smart web-based platform developed to streamline the process of monitoring, booking, and managing parking slots across different cities. It is designed to serve both users and administrators with efficiency and clarity.

Purpose

To reduce parking chaos by providing a centralized, real-time slot monitoring system — accessible anytime, from anywhere.

Core Features

  • City-wise slot tracking and booking
  • Real-time display of available (vacant) slots only
  • Dynamic slot creation from file input
  • Admin-friendly code structure

Tech Stack

  • Frontend: HTML, CSS (Dark theme with red & white contrast)
  • Backend: Python (Flask framework)
  • Storage: .txt files
  • Rendering: Jinja2 templates
